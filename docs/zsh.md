# zsh

```bash
# Kill process with percol
$ ppkill

# Get PID with percol
$ ppgrep

# Git checkout
$ git-use

# Git checkout ( list include remote repo branches)
$ git-all-use
```

* `ctrl + r` : search history with percol
* `ctrl + u` : Reset Command
* `cd -` : back to old current directory
* `cd -<TAB>` : select back directory on histroy
